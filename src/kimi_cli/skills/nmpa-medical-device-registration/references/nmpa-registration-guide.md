| 中国食品药品检定研究院（中检院，NIFDC） | 标准品、注册检验技术指导、部分产品检验 |

## 产品分类判定

### 判定要素

依据《医疗器械分类规则》（总局令第15号），分类需综合以下要素：

1. **结构特征**：有源 / 无源 / 植入 / 接触人体
2. **使用形式**：诊断、治疗、监护、输送、避孕等
3. **使用状态**：使用时是否接触人体、接触部位（皮肤/黏膜/腔道/组织/血液/中枢）
4. **作用时间**：暂时使用（≤24h）/ 短期使用（24h–30d）/ 长期使用（>30d）
5. **失控后危害程度**：轻微 / 中等 / 严重

### 管理类别

| 类别 | 风险程度 | 管理方式 | 审批机关 |
|------|----------|----------|----------|
| I类 | 低 | 备案 | 设区市药监局（境内）；NMPA（进口） |
| II类 | 中 | 注册 | 省级药监局（境内）；NMPA（进口） |
| III类 | 高 | 注册 | NMPA（境内/进口） |

### 分类界定流程

当《医疗器械分类目录》无法直接覆盖或存在争议时：

1. **自查**：先查《医疗器械分类目录》及动态调整文件
2. **预沟通**：通过 CMDE 分类界定系统或省级药监部门咨询
3. **正式申请**：境内产品向所在地省级药监局申请；进口产品向 NMPA 申请
4. **等待复函**：通常 20 个工作日内出具分类界定意见
5. **按意见执行**：以监管机关出具的分类界定意见为准

### IVD 分类特别提示

体外诊断试剂按风险分为 I、II、III 类，需参考《体外诊断试剂分类子目录》（2013版及后续调整）：

- III类：与血型/组织配型、病原体（高风险）、遗传性疾病、恶性肿瘤、伴随诊断相关
- II类：除III类外的其他用于疾病诊断、治疗监测等
- I类：微生物培养基、样本处理产品等低风险产品

## 注册与备案路径

### 境内产品路径

```
I类   → 产品备案（设区市药监局）+ 生产备案
II类  → 产品注册（省级药监局）+ 生产许可（省级药监局）
III类 → 产品注册（NMPA）+ 生产许可（省级药监局）
```

### 进口产品路径

```
I类   → 向 NMPA 备案
II类  → 向 NMPA 注册
III类 → 向 NMPA 注册
```

### 进口产品境内代理人

依据《医疗器械监督管理条例》，进口医疗器械必须由**境内企业法人**作为代理人，承担以下责任：

- 协助境外注册人履行义务
- 配合药监部门监督检查
- 承担医疗器械不良事件监测与报告
- 承担产品召回责任
- 负责售后服务

**代理人资质要求**：在中国境内设立的企业法人，具有与所代理产品相适应的质量管理能力。

### 特殊审评通道

| 通道 | 条件 | 优势 |
|------|------|------|
| 创新医疗器械特别审查 | 拥有产品核心技术发明专利/专利申请；具有显著临床应用价值；产品国内首创 | 优先审评、专人负责、可早期沟通 |
| 优先审批程序 | 列入国家科技重大专项/重点研发计划；诊断/治疗罕见病、恶性肿瘤、儿童疾病、老年人特有或多发疾病；临床急需 | 优先审评审批 |
| 应急审批 | 突发公共卫生事件急需 | 加速审评 |

## 申报资料清单

境内II类/III类及进口产品注册申报资料依据 2021年第121号公告（eRPS），共包含以下模块：

### 模块一：监管信息

| 资料项 | 说明 |
|--------|------|
| 申请表 | 在 NMPA 网上办事大厅填写并导出 |
| 术语/缩写词列表 | 产品相关术语解释 |
| 产品列表 | 同一注册单元内所有型号/规格 |
| 关联文件 | 营业执照、创新/优先审批证明文件等 |
| 申报前与监管机构的联系情况和沟通记录 | 如 pre-submission 会议纪要 |

### 模块二：综述资料

| 资料项 | 核心内容 |
|--------|----------|
| 章节目录 | — |
| 概述 | 产品名称、管理类别、分类编码、预期用途、结构组成、工作原理、型号规格 |
| 产品描述 | 详细技术特征、主要原材料、关键工艺 |
| 研发历程 | 设计开发过程概述 |
| 一般信息 | 包装、灭菌、有效期、储存条件 |
| 上市历史 | 已上市国家/地区、不良事件、召回情况（进口产品必须） |

### 模块三：非临床资料

| 资料项 | 核心内容 |
|--------|----------|
| 产品性能研究 | 技术要求中各指标的确定依据、研究资料 |
| 生物相容性评价 | 按照 GB/T 16886 系列进行 |
| 灭菌/消毒工艺研究 | 灭菌验证、残留毒性、效期验证 |
| 稳定性研究 | 货架有效期、使用稳定性、运输稳定性 |
| 软件研究 | 软件描述文档、网络安全文档（如适用） |
| 生物学特性研究 | 如适用 |
| 动物试验 | 必要时提供 |
| 风险分析资料 | 按 YY/T 0316 / ISO 14971 进行 |
| 临床试验资料 | 或临床评价资料 |
| 产品技术要求 | 即注册产品标准，含性能指标和检验方法 |
| 注册检验报告 | 由具备资质的检验机构出具 |
| 说明书和标签样稿 | 符合《医疗器械说明书和标签管理规定》（总局令第6号） |

### 模块四：临床评价资料

| 资料项 | 说明 |
|--------|------|
| 临床评价路径选择 | 免于临床评价 / 同品种比对 / 临床试验 |
| 临床评价报告（CER） | 按 CMDE 模板编写 |
| 临床试验方案与报告 | 如开展临床试验 |

### 模块五：产品说明书和标签样稿

- 产品名称、型号、规格
- 注册人/生产企业信息
- 结构组成、主要性能、适用范围
- 禁忌症、注意事项、警示
- 安装使用说明、维护保养
- 标签样稿

### 模块六：质量管理体系文件

| 资料项 | 说明 |
|--------|------|
| 生产制造信息 | 生产工艺流程图、关键工序、特殊过程 |
| 质量管理体系文件 | 质量手册、程序文件、记录清单 |
| 质量管理体系核查文件 | 自查报告、内审/管审记录 |

### 备案资料（I类）

I类备案相对简化，通常需：

1. 备案表
2. 产品风险分析资料
3. 产品技术要求
4. 产品检验报告（自检或委托检验）
5. 临床评价资料
6. 产品说明书及标签样稿
7. 生产制造信息
8. 质量管理体系文件
9. 符合性声明

## 临床评价路径

### 路径一：免于临床评价

适用条件：
- 列入《免于临床评价医疗器械目录》
- 产品与目录内产品具有等同性
- 可证明产品安全性、有效性可通过非临床证据支持

需提交：
- 免于临床评价对比说明
- 与目录产品的对比表（工作原理、结构组成、性能要求、适用范围、灭菌方式、包装等）
- 非临床证据（性能研究、生物学评价、稳定性研究等）

### 路径二：同品种比对（实质等同）

适用条件：
- 申报产品与已上市同品种产品在基本原理、结构组成、性能要求、适用范围、使用方法等方面相同或无重大差异
- 差异可通过非临床研究或临床经验证明安全有效

需提交：
- 同品种医疗器械选择依据
- 申报产品与同品种产品对比表
- 同品种产品临床数据（文献、不良事件、临床经验）
- 差异分析与支持证据
- 临床评价报告

**同品种数据来源**：
- 公开发表的文献
- 已上市同品种产品的临床使用数据
- 上市后监督数据（不良事件、召回）
- 境外临床数据（需符合接受要求）

### 路径三：临床试验

适用条件：
- 不在免临床目录内
- 无法找到合适同品种产品
- 同品种比对无法充分证明安全有效性

需提交：
- 临床试验方案
- 伦理委员会批准文件
- 知情同意书样本
- 临床试验报告
- 统计分析报告
- 原始数据清单

**临床试验机构要求**：
- 在 NMPA 备案的医疗器械临床试验机构
- 具备相应专业和设施
- 主要研究者具备资质

### 临床评价报告（CER）结构

```
1. 产品描述
2. 临床评价范围
3. 临床评价路径及选择理由
4. 同品种产品信息（如适用）
5. 临床数据识别、评估与分析
6. 差异分析（如适用）
7. 结论
8. 参考文献
```

## 注册检验与检测

### 注册检验类型

1. **注册检验（型式检验）**：注册前必须完成，由具备资质的检验机构出具
2. **补充检验**：发补时根据审评意见补充
3. **变更检验**：涉及产品技术要求变更时

### 检验机构

- 中国食品药品检定研究院（NIFDC）
- 各省医疗器械检验所
- 其他经 NMPA 认可的第三方检验机构

### 技术要求主要内容

产品技术要求应至少包括：

1. 产品型号/规格及其划分说明
2. 性能指标
3. 检验方法
4. 附录（如电气安全、电磁兼容、生物相容性、软件等）

### 常见检测标准

| 标准 | 适用范围 |
|------|----------|
| GB 9706.1 / YY 0505 | 医用电气设备安全与电磁兼容 |
| GB/T 16886 系列 | 医疗器械生物学评价 |
| YY/T 0316 | 医疗器械风险管理 |
| YY/T 0664 | 医疗器械软件 软件生存周期过程 |
| GB/T 25000.51 | 软件产品质量要求与评价 |
| YY/T 1843 | 医用电气设备网络安全 |

### 检测注意事项

- 检验样品应为拟上市产品的典型型号
- 电气安全、EMC 检测需覆盖所有型号/配置或提供典型性说明
- 生物学评价可引用原材料供应商数据，但需进行风险评估
- 软件/网络安全文档需与技术要求中软件版本一致

## 质量体系与现场核查

### 医疗器械生产质量管理规范（GMP）

依据《医疗器械生产质量管理规范》及附录，企业需建立覆盖以下方面的质量体系：

1. 机构与人员
2. 厂房与设施
3. 设备
4. 文件管理
5. 设计开发
6. 采购
7. 生产管理
8. 质量控制
9. 销售和售后服务
10. 不合格品控制
11. 不良事件监测、分析和改进

### 现场核查触发条件

| 情形 | 核查机关 | 说明 |
|------|----------|------|
| 境内II类/III类首次注册 | 省级药监局 | 注册审评时同步或依申请开展 |
| 生产许可证核发/延续 | 省级药监局 | 必查 |
| 进口产品注册 | NMPA 必要时境外核查 | 通常不进行现场核查，除非需要 |
| 投诉举报、抽检不合格 | 药监部门 | 有因核查 |

### 现场核查重点

- 设计开发文档完整性与可追溯性
- 采购控制（供应商审计、原材料检验）
- 生产过程控制与记录
- 检验设备校准与检验记录
- 产品放行与留样
- 不良事件监测体系
- 说明书与标签管理

## 审评流程与发补应对

### 审评时限（法定）

| 事项 | 时限 |
|------|------|
| 境内II类注册 | 60个工作日 |
| 境内III类注册 | 90个工作日 |
| 进口II类/III类注册 | 90个工作日 |
| 创新医疗器械 | 优先审评，时限适当缩短 |

注：发补时间、专家咨询时间、现场核查时间不计入审评时限。

### 审评流程

```
申请受理 → 技术审评 → 发补（如需）→ 补充资料审评 → 行政审批 → 发证
```

### 常见发补原因

| 类别 | 常见发补点 |
|------|------------|
| 综述资料 | 产品描述不清晰、型号规格划分不合理、预期用途表述不准确 |
| 非临床资料 | 性能指标依据不足、检验方法不明确、稳定性研究不完整、风险管理不到位 |
| 临床资料 | 临床评价路径选择不当、同品种对比差异未充分论证、临床试验设计缺陷 |
| 技术要求 | 指标低于国标/行标、检验方法不可操作、未覆盖全部型号 |
| 说明书标签 | 禁忌症/注意事项缺失、夸大疗效、未标注必要警示 |
| 质量体系 | 设计开发记录不完整、检验记录缺失、供应商审计不足 |

### 发补应对原则

1. **逐条响应**：每条发补意见单独回复，引用资料页码
2. **证据充分**：补充试验数据、文献、说明或修订文件
3. **一致性**：确保回复资料与已提交资料无矛盾
4. **及时性**：在规定时限内（通常1年）提交补充资料
5. **必要时沟通**：复杂问题可申请审评沟通会议

## 上市后管理

### 生产许可

- II类、III类医疗器械生产企业须取得《医疗器械生产许可证》
- I类生产企业办理生产备案
- 生产许可证有效期5年，届满前6个月申请延续

### 注册变更

| 变更类型 | 示例 | 办理方式 |
|----------|------|----------|
| 许可事项变更 | 产品名称、型号规格、结构组成、适用范围、技术要求、生产地址实质性变更 | 变更注册申请 |
| 登记事项变更 | 注册人名称、住所、代理人名称、生产地址文字性变更 | 登记事项变更 |

### 延续注册

- 注册证有效期5年
- 应在有效期届满前6个月申请延续
- 需提交延续注册申请表、原注册证、产品技术要求、上市后评价资料等
- 逾期需按首次注册办理

### 不良事件监测与召回

- 建立医疗器械不良事件监测体系
- 发现严重不良事件应及时报告
- 产品存在安全隐患时，主动召回并报告

### 年度自查与报告

- 医疗器械注册人、备案人应定期开展质量管理体系自查
- 按要求向药监部门提交自查报告

## 特殊产品注意事项

### 软件医疗器械（SaMD）

- 需明确软件安全性级别（A/B/C级，按 YY/T 0664 / IEC 62304）
- 提供软件描述文档、软件版本命名规则
- 网络安全文档（按 YY/T 1843）
- 人工智能/机器学习算法需额外说明训练数据、算法性能、更新策略

### AI/机器学习医疗器械

- 需符合《人工智能医疗器械注册审查指导原则》
- 提供算法研究资料（训练集、调优集、测试集）
- 算法性能评价（准确率、敏感性、特异性、AUC等）
- 泛化能力证明
- 软件更新控制程序

### 有源医疗器械

- 电气安全：GB 9706.1 及专用安全标准
- 电磁兼容：YY 0505 / GB 9706.102
- 环境试验
- 可用性工程（按 YY/T 1474 / IEC 62366）

### 无源植入器械

- 材料表征与生物相容性
- 疲劳/磨损试验
- 动物试验（必要时）
- 灭菌验证与有效期

### 体外诊断试剂（IVD）

- 参考区间确定
- 分析性能评估（精密度、准确度、检出限、线性、干扰、稳定性）
- 临床性能评估
- 校准品/质控品溯源

### 组合产品

- 药品+医疗器械组合产品需明确主性质
- 向 NMPA 申请属性界定
- 分别满足药品/器械法规要求

## 常见错误与风险

### 高风险错误

| 错误 | 后果 |
|------|------|
| 产品分类错误 | 注册路径错误，可能被退审或处罚 |
| 预期用途超范围 | 虚假宣传、超范围经营风险 |
| 临床评价路径选择不当 | 发补、退审、要求补做临床试验 |
| 技术要求指标低于国标 | 无法通过技术审评 |
| 检验样品非典型型号 | 检验结论不被认可 |
| 说明书标签不合规 | 被要求修订，影响上市进度 |
| 质量管理体系不满足GMP | 现场核查不通过 |

### 合规建议

1. 注册前进行充分的法规调研和分类确认
2. 尽早与 CMDE 或省级药监部门进行 pre-submission 沟通
3. 选择有经验的 CRO、检验机构和法规顾问
4. 建立完善的设计开发文档和设计历史文件（DHF）
5. 上市后持续监测不良事件和产品性能
6. 关注法规更新（目录调整、指导原则、标准变化）
